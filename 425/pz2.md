## Практическое задание

### Цель

Закрепить базовые навыки работы с файловой системой в **Node.js**: чтение файла, обработка его содержимого и вывод результата в консоль.

### Задание (шаги)

1. Создайте папку проекта и инициализируйте Node.js-проект:

   ```bash
   npm init -y
   ```

2. В папке проекта создайте текстовый файл `data.txt`.

3. Запишите в `data.txt` несколько строк текста, например:

   ```
   Node.js is great
   I like programming
   This is a simple task
   ```

4. Создайте файл `app.js`.

5. В `app.js`:
   - подключите встроенный модуль `fs`;
   - прочитайте содержимое файла `data.txt`;
   - выведите содержимое файла в консоль.

6. Дополнительно обработайте текст:
   - посчитайте количество строк в файле;
   - выведите это число в консоль.

7. Запустите программу:

   ```bash
   node app.js
   ```

Ожидаемый результат:

- содержимое файла выводится в консоль;
- отдельно выводится количество строк.

### Подсказки по ключевым частям

**Подключение модуля `fs`:**

```js
const fs = require("fs");
```

**Чтение файла:**
Вы можете использовать:

- `fs.readFile()` (асинхронно)
  или
- `fs.readFileSync()` (синхронно)

Пример синхронного чтения:

```js
const text = fs.readFileSync("data.txt", "utf-8");
```

**Разделение текста на строки:**

```js
const lines = text.split("\n");
```

**Подсчёт строк:**

```js
console.log(lines.length);
```

### Что проверить перед отправкой (чек-лист)

Перед сдачей задания убедитесь, что:

- файл `data.txt` находится в той же папке, что и `app.js`;
- программа корректно запускается через `node app.js`;
- содержимое файла выводится в консоль;
- количество строк считается правильно;
- в коде нет ошибок выполнения;
- код читаемый (понятные названия переменных).

### Советы по улучшению работы

Если базовое задание выполнено, можно усложнить программу:

1. **Добавить обработку ошибок** (например, если файл не найден).
2. **Посчитать количество слов** в файле.
3. **Посчитать количество символов**.
4. **Сделать вывод более информативным**, например:

   ```
   Количество строк: 3
   Количество слов: 10
   ```

5. Попробовать реализовать **и синхронную, и асинхронную версию чтения файла** и сравнить их.

При выполнении задания старайтесь писать код аккуратно и структурировано. Это поможет быстрее находить ошибки и упростит дальнейшее расширение программы.
