## Практическое задание

**Тема:** jQuery: селекторы, DOM, содержимое, атрибуты, стили, размеры, события

## Цель

- Научиться применять селекторы jQuery для точного выбора элементов
- Закрепить навигацию по DOM
- Освоить работу с содержимым (`text`, `html`, `val`)
- Понять разницу между `attr`, `prop`, `data`
- Использовать классы и `.css()` для управления стилями
- Работать с размерами и позиционированием
- Реализовать обработку событий и делегирование

## Задание (шаги)

### 1. Создать базовую разметку

- Создать HTML-страницу со структурой:
  - Заголовок `<h1>`
  - Поле ввода (`input`)
  - Кнопка «Добавить задачу»
  - Список задач `<ul id="list">`
  - Блок для вывода информации (`#output`)
  - Контейнер-обёртка (для работы с позиционированием)

### 2. Добавление элементов (селекторы + события)

- При клике на кнопку:
  - Получить значение из input через `.val()`
  - Добавить новый `<li class="item">` в список через `.append()`
  - Очистить input

- Использовать селекторы:
  - по id (`#list`)
  - по классу (`.item`)

### 3. Делегирование событий

- Назначить обработчик через `.on("click", ".item", ...)`
- При клике на элемент списка:
  - Удалить класс `active` у всех элементов
  - Добавить класс `active` текущему (`$(this)`)

### 4. Работа с DOM (навигация)

- При клике на элемент:
  - Найти родителя через `.parent()`
  - Найти соседей через `.siblings()` и изменить их стиль
  - Найти дочерние элементы (если есть) через `.find()`

### 5. Работа с содержимым

- Добавить кнопку «Показать текст»
  - Получить текст выбранного элемента через `.text()`

- Добавить кнопку «Изменить HTML»
  - Вставить HTML внутрь элемента через `.html()`

- Добавить кнопку «Очистить список»
  - Использовать `.empty()`

### 6. Работа с атрибутами

- При добавлении элемента:
  - Добавить `data-id` через `.attr()` или `.data()`

- Реализовать:
  - Получение `data-id`
  - Изменение атрибута
  - Удаление атрибута через `.removeAttr()`

### 7. Работа со стилями

- При клике:
  - Использовать `.addClass()`, `.removeClass()`, `.toggleClass()`

- Добавить кнопку:
  - Изменить стиль через `.css()`

- Проверить наличие класса через `.hasClass()`

### 8. Размеры и позиционирование

- Добавить кнопку:
  - Получить `.width()` и `.outerWidth()`

- Добавить кнопку:
  - Получить `.offset()` и `.position()`

- Вывести значения в `#output`

### 9. События клавиатуры

- Отслеживать `keydown` на input
- Выводить `event.key` в интерфейс

### 10. Работа с формой

- Создать форму
- При submit:
  - Использовать `event.preventDefault()`
  - Вывести введённое значение

### 11. Дополнительно

- Использовать:
  - `.one()` — кнопка, которая срабатывает один раз
  - `.off()` — удалить обработчик

## Подсказки по ключевым частям

- Используй `$(this)` внутри обработчиков — это текущий элемент
- Для динамических элементов всегда используй делегирование
- `.text()` — безопасно, `.html()` — вставляет HTML
- Для чекбоксов используй `.prop()`, не `.attr()`
- CSS-свойства в jQuery пишутся в `camelCase`
- `.find()` ищет глубоко, `.children()` — только прямых потомков
- `.offset()` — относительно страницы
- `.position()` — относительно родителя
